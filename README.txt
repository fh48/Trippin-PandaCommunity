Welcome to trippin' panda! :)

To run tpanda, you'll need to:

1. Navigate to tpanda/frontend and npm install
2. Navigate to tpanda/backend and npm install
3. Run "node app.js"
4. Navigate to localhost:/3000
5. Get trippin' !!

API DOCUMENTATION
Details on API design are documented in the
API_doc.html file in the main folder.
